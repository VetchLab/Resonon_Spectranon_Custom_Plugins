Band Summary Auto Version V2 Plugin for Spectronon

Description

The Band Summary Auto Version V2 is an advanced CubePlugin for Spectronon software, designed for automatic calculation of comprehensive statistics across all bands in a hyperspectral datacube.

Features

Calculates comprehensive statistics for each wavelength (minimum, maximum, percentiles, mean, standard deviation, variance, skewness, and kurtosis).
Exports the calculated statistics to a CSV file.
Option to exclude zero values from calculations.
Allows selection of output directory for the CSV file.

Installation

Download BandSummaryAutoV2.py.
Place it in the %localappdata%\SpectrononPro3\user_plugins\cube\user\ directory.

Usage

Open Spectronon and load a datacube.
Go to the User plugins menu and select Band Summary Auto Version V2.
Set your preferences in the plugin interface.
Click Run to execute. The plugin will process each wavelength and save the results.

Output

A CSV file named band_summary_statistics.csv containing statistics for each wavelength.
